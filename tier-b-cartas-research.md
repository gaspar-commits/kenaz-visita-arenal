# Resumen ejecutivo — para decidir upsell carta digital

## Top 3-5 candidatos donde carta digital Kenaz = upsell ganador

| # | Negocio | Razón clave |
|---|---------|-------------|
| 1 | **Mosselhuis** | Plantilla qrcarta.com + sin web propia + signature monoproducto (mejillones) ideal para mostrar visualmente |
| 2 | **Las Olas** | PDF descargable + web plantilla + signature claro (arroces/paellas) que pide foto pro |
| 3 | **Morena Restobar** | Carta gigante mal presentada en sitios genéricos + web abandonada |
| 4 | **Hydra** | (mid) Tiene identidad pero WP/Astra — upsell de "decente" a "premium" |
| 5 | **Semsa Beach** | (mid-bajo) ya tiene carta-QR propia, upsell menos dramático |

## Descartados para carta digital (incluir en kit Tier B sin carta)

| Negocio | Razón |
|---------|-------|
| Suitopia Sol y Mar | Corporativo Estimar — ciclo largo, decisión no local |
| Hotel Solymar | Mismo grupo Estimar — mismo problema |
| Pastelería Av Europa 1 | Mostrador, no carta tradicional. Verificar in-situ |
| Heladería Peca2 | Mostrador, sabores rotativos. Web/IG sí, carta no |
| Amara | Heladería-cafetería de mostrador (la activa) o cerrada (la otra) |

## Hallazgos críticos para la visita martes 13-may

1. **Mosselhuis es el caso de oro del Tier B** — qrcarta.com white-label + monoproducto + sin web = manual perfecto del antes/después. Llevar mockup conceptual si se puede.
2. **Las Olas** es candidato sólido pero el grupo del paseo suele ser conservador — preparar argumento "tus arroces merecen foto pro, no PDF".
3. **Morena vs Las Olas:** Las Olas tiene signature claro, Morena no. Las Olas convierte mejor.
4. **Hydra zona dudosa** — confirmar in-situ que el local sigue activo en Av. Rosa de los Vientos 16. Si está, tiene buen producto pero el upsell de carta es secundario (web decente).
5. **Amara ambigüedad** — hay un Amara cerrado (Faboaranzabal) y un Amara heladería activo. No confundirlos.
6. **Hoteles (Suitopia + Solymar)** — descartar carta digital de la propuesta. Si entra, va por canal corporativo Estimar, no local.
7. **Pastelería Av Europa 1** — research no devolvió nombre claro. Verificar IN-SITU el martes el nombre exacto y si tienen carta de cafetería/desayunos.

## Recomendación de kit Tier B

- **Kit completo (web + carta + foto + IG + Google):** Mosselhuis, Las Olas, Morena
- **Kit completo sin carta (web + foto + IG + Google):** Pastelería Av Europa 1, Peca2, Amara heladería
- **Kit upgrade (carta premium + ajuste web):** Hydra, Semsa Beach
- **Descartar Tier B (corporativo):** Suitopia, Solymar — pasarlos a track de cuentas corporativas separado
